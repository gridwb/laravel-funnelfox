## Installation

1. Install the package
    ```bash
    composer require gridwb/laravel-funnelfox
    ```

2. Publish the configuration file
    ```bash
    php artisan vendor:publish --tag="funnelfox-config"
    ```

3. Add environment variables
    ```env
   TEST=test
    ```
